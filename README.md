## 2. How to run?

> ### docker-compose build

    docker-compose build

    docker-compose up -d

    OR

    . dc.sh

OR

Just run web only

Move the ui directory and

    cd ui/

> ### run storybook on your local

    npm run storybook

> ### run next js on your local

    npm run dev

> ### run npm build

    npm run build
