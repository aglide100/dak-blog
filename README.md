# personel-blog

### This repo is make simple web blog

## 1. purpose

writing some code or project at own blog

## 2. How to run?

Move the ui directory and

    cd ui/

> ### run storybook on your local

    npm run storybook

> ### run next js on your local

    npm run dev

> ### run npm build

    npm run build
